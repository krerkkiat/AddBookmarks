# Add Bookmarks

A thin [cpdf](https://www.coherentpdf.com/index.html) wrapper to add bookmarks from a CSV file to a PDF file.

## Why?

This little tool is built to help me add bookmarks to [Dr. Supee's Dhamma books](https://www.ajsupee.com/showbooks).
The source of the data is available [here](https://docs.google.com/spreadsheets/d/1GG6_-yZCsTcoetXN9Qz9HIvgvMZatOPktbV44VvmwwI/edit?usp=sharing).
